# 🎮 Advanced Hangman Game

A fun and interactive **Command-Line Hangman Game** built with **Python** as part of the CodeAlpha Internship Program. The game includes multiple difficulty levels, ASCII hangman graphics, hints, input validation, and replay functionality.

📂 GitHub Repository:  
https://github.com/HafsaRahman05/Hangman_Game-CodeAlpha-

---

## 📌 Features

- 🎯 Three difficulty levels:
  - Easy
  - Medium
  - Hard
- 🎨 ASCII Hangman visual representation
- 💡 One-time hint system
- ✅ Input validation
- 🔄 Replay game option
- 📝 Tracks guessed letters
- 🎉 Win and lose conditions
- 💻 Beginner-friendly Python project

## 🎮 How to Play

1. Launch the game.
2. Select a difficulty level:
   - Easy
   - Medium
   - Hard
3. Guess one letter at a time.
4. Type `hint` to receive a clue (available once per game).
5. Avoid too many wrong guesses.
6. Guess the complete word before the hangman is fully drawn.

## 💡 Game Features Explained

### ASCII Hangman Graphics
The hangman figure is drawn progressively after each incorrect guess.

### Hint System
Players can request one hint per game by typing:

```text
hint
```

### Input Validation
The game prevents:
- Empty input
- Multiple characters
- Numbers
- Symbols
- Repeated guesses
